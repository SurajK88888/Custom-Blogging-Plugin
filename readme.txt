A Production-Level WordPress Blog Management Plugin with Email Marketing, Ad Management, Analytics, Role-Based Access Control, Modern UI, and Fully Customizable Design.

== Description ==

Custom Blog Pro (CBP) is a fully-featured, SaaS-like blog management plugin for WordPress. It features a modern card-based admin dashboard, built-in email marketing, advertisement management, in-depth analytics, and a fully customizable frontend UI that matches your theme.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/custom-blog-pro` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Settings->Custom Blog Pro screen to configure the plugin.
